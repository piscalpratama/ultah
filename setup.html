Setup Wajah Lia

Halaman ini untuk mendaftarkan wajah Lia ke sistem pengenalan.
Lakukan sekali saja — hasilnya disimpan ke faces.json.

1
Upload foto Lia (3–8 foto)

Pilih foto yang jelas wajahnya, berbeda ekspresi dan sudut. Makin beragam fotonya, makin akurat pengenalannya.

🌸
Klik untuk pilih foto Lia
PNG, JPG — bisa lebih dari satu
2
Proses pengenalan wajah

Sistem akan membaca wajah dari setiap foto dan membuat descriptor unik. Ini butuh beberapa detik.

3
Download & upload ke GitHub

Download file faces.json lalu upload ke root repo GitHub kamu di sebelah index.html.

Langkah selanjutnya:

  1. Download faces.json dari tombol di atas
  2. Buka repo GitHub kamu
  3. Upload faces.json ke root folder (sejajar index.html)
  4. Buka website utama — sudah siap! 🎉

Siap dipakai!

File faces.json sudah kamu download.
Upload ke GitHub, dan website ulang tahun Lia sudah bisa mengenali wajahnya.
